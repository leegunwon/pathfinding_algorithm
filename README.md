# pathfinding_algorithm
---
두 개의 로봇이 그리드 월드 내에서 충돌없이 목적지에 도착하는 알고리즘

각 로봇의 시작 지점은 동일하며 로봇의 목적지는 총 6개 존재한다.
그리드 맵은 아래와 같이 구성되어 있음 [1은 장애물을 의미하고 0은 진행 가능한 동선을 의미한다] 


1     0     1     0     1

0     0     0     0     0

1     0     1     0     1

0     0     0     0     0

1     0     1     0     1
